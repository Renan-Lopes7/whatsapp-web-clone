# WhatsApp Web clone

Clone da interface do WhatsApp Web desenvolvido com HTML e CSS puro, design fiel à versão original.
Projeto focado em práticas de front-end, posicionamento CSS e estruturação semântica.


## Tecnologias utilizadas
HTML5

CSS3

### Funcionalidades
Sidebar com perfil e notificações
Lista de conversas com indicadores de mensagens não lidas
Projeto desenvolvido como parte de um curso de HTML/CSS
